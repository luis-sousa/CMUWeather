# CMUWeather
Aplicação de tempo desenvolvida em Android.

CMUWeather é uma aplicação móvel para dispositivos Android que pretende apresentar previsões metereológicas para diversos locais. O trabalho tem como base uma API fornecida pelo docente a biblioteca CMUWeatherLib.
#Principais Funcionalidades
- Apresentação da temperatura para uma ou várias cidades.
- Previsões para os próximos 5 dias
- Informacões detalhadas: humidade, temperatura, precipitação, vento
- Sistema de preferências: aplicação onde pode ser definido o local a apresentar na Widget , a velocidade do vento (KM/H ou Miles) e a unidade de temperatura (Celsius ou Fahrenheit).
